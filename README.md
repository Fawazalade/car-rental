# D RENTALS - Premium Car Rental Website

A sleek, professional, and fully responsive car rental website built from scratch. D RENTALS provides users with an elegant interface to browse, select, and rent premium vehicles.

## ✨ Features

- **Hero Section**: Stunning header with car search/booking form
- **Vehicle Gallery**: Browse diverse vehicle categories (Cars, SUVs, Vans, Electric)
- **Location Finder**: Search for rental cars in your area
- **Interactive Car Slider**: Swipeable carousel showcasing premium vehicles with detailed specs
- **Dynamic Pricing**: Real-time price updates based on selected vehicle ($225 - $625/day)
- **Blog Section**: "Stories Behind the Wheel" - customer experiences and travel stories
- **Brand Banners**: Trusted partner logos display
- **Mobile App Download**: Links to Apple App Store and Google Play Store
- **Newsletter Signup**: Email subscription for updates
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop
- **Smooth Animations**: Engaging scroll reveal animations

## 🛠️ Tech Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with modern CSS features
- **JavaScript (Vanilla)**: Interactive functionality
- **Swiper.js**: Premium car carousel/slider
- **ScrollReveal**: Scroll-triggered animations
- **Remix Icons**: Icon library for UI elements

## 🚀 Getting Started

1. **Clone the repository**:
   
```
bash
   git clone <repository-url>
   
```

2. **Navigate to the project directory**:
   
```
bash
   cd car-rental
   
```

3. **Open in browser**:
   Simply open `index.html` in your preferred web browser, or use a local server:
   
```
bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   
```

4. **Customize**:
   - Modify `index.html` for content changes
   - Update `styles.css` for styling adjustments
   - Edit `main.js` for functionality modifications
